# Vishal Deshmukh — Personal Portfolio

A responsive personal portfolio website showcasing my projects, technical skills, and certifications as an aspiring Web Development / Python Development professional.

🔗 **Live Site:** https://vishal3399.github.io/V-portfolio/

## Features
- Responsive design compatible with desktop, tablet, and mobile devices
- Sections for About, Education, Skills, Projects, Certificates, and Contact
- Dedicated Projects page showcasing all work with live demo links
- Certificate gallery with images and issuing organizations
- Downloadable resume (PDF) directly from the site
- Contact section with multiple channels (email, phone, LinkedIn, GitHub, WhatsApp)

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript

## What I Learned
- Building a fully responsive multi-page site from scratch using core HTML/CSS/JS
- Structuring content across multiple pages (home + dedicated projects page) for better navigation
- Presenting projects, certifications, and resume in a way that's clear and recruiter-friendly
- Deploying and maintaining a live site via GitHub Pages
